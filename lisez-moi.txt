Gh0st est un script qui automatise la parti footprinting avec plusieurs scripts kali et perso un tout en un quoi ! :) pour en faire le meilleurs script d'automatisation du footprinting.

Mettre kali linux à jour avec cette commande : apt-get update && apt-get upgrade && apt-get dist-upgrade 

Mettre le contenu du fichier opt dans /opt et le contenu du dossier application dans usr/share/application un raccourci sera fait dans le menu kali /top 10

A installer pour le bon fonctionnement curl : 
sudo apt-get install php5-curl
sudo apt-get install curl libcurl3 libcurl3-dev
